# Calculator_js
A fully functional, browser-based calculator built as part of **The Odin Project** Foundations curriculum. This project focuses on complex logic, DOM manipulation, and clean UI design.

[Live Demo Link](INSERT_YOUR_VERCEL_OR_NETLIFY_LINK_HERE)

## 🚀 Features
- **Basic Arithmetic:** Supports Addition, Subtraction, Multiplication, and Division.
- **Float Support:** Handles decimal numbers and prevents multiple decimal points.
- **Keyboard Support:** Full support for number keys and operators via keyboard.
- **Error Handling:** Custom message for "Division by Zero" (the universe doesn't explode!).
- **Clear & Delete:** Option to clear the entire screen or delete the last digit entered.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure for the calculator body.
- **CSS3:** Flexbox and Grid for a responsive, modern layout.
- **JavaScript (Vanilla):** Logic for mathematical operations and UI updates.

## 🧠 What I Learned
- **Logic Sequencing:** Managing "First Number," "Operator," and "Second Number" variables.
- **Event Delegation:** Efficiently handling button clicks using event listeners.
- **Precision Handling:** Solving the classic JavaScript `0.1 + 0.2` floating-point math issue.
- **UI/UX:** Designing a clean, dark-mode interface that mimics modern smartphone calculators.
